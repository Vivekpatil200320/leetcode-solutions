My data structures & algorithms practice log, in Python.

## Approach

- **Cadence:** 3–4 problems per week, solved consistently rather than crammed.
- **Roadmap:** Following the [NeetCode 75](https://neetcode.io/practice) list — the folder structure below mirrors its categories, in the order the roadmap builds on itself (hashing before two pointers, trees before graphs, 1-D DP before 2-D).
- **Language:** Python. Each solution notes its time/space complexity and a one-line intuition, so revisiting a problem months later doesn't mean re-deriving it from scratch.
- **Rule:** No copy-pasting an editorial. If I can't solve it, I study the pattern, close the tab, and implement it from memory.
